背景自定义必须升级

背景不能只有颜色。需要支持完整背景系统。

建议 page JSON 支持：

{
  "background": {
    "type": "layered",
    "layers": [
      {
        "type": "color",
        "value": "#020617"
      },
      {
        "type": "image",
        "asset_key": "background.training.main",
        "fit": "cover",
        "position": "center",
        "opacity": 0.85
      },
      {
        "type": "gradient",
        "from": "#020617",
        "to": "#0F172A",
        "direction": "vertical",
        "opacity": 0.55
      },
      {
        "type": "overlay",
        "value": "#000000",
        "opacity": 0.25
      }
    ]
  }
}

其中 image.asset_key 必须通过 manifest / resource bundle 解析，不允许 QML 直接写死 assets/xxx.png 路径。
